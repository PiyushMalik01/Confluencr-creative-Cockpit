05

Generated image prompts, ready to run

The prompts above were lifted into Nano Banana Pro and Flux Kontext as a sanity check. Three things were true under load.

Concept 1 · Aspirational hero lands first time. Flux Kontext renders the yellow-on-black gig-poster aesthetic cleanly when given a real product photo as the conditioning image. No regen needed for the Amazon 1:1.
Concept 3 · Rational infographic is best run on Ideogram. The legibility of the four corner callouts in tall Inter is the differentiator versus Flux, which still wobbles on short on-image text under 12pt equivalent. Route by shot type in Step 6.
Concept 2 · Emotional lifestyle needs a hand-pick. The room-corner moment is the right vibe but the AI occasionally hallucinates a different chair or a different tee colour. The auto-fix loop helps but a human pick across 3-4 candidates is faster than auto-regen.
06

Commentary — what worked, what did not, what I would iterate

✓  What worked

Trifecta enforcement on the angle picker. The single highest-signal mechanism in the cockpit. First time I ran the system with no enforcement, gpt-5.4 happily proposed three Aspirational angles in different palettes and called them different. The picker refusing to advance unless ≥ 2 trifecta families are picked changed the output meaningfully.
Evidence array on the StyleReport schema. Forcing the LLM to ground every claim to a specific image index killed the "competitors use heavy grain" hallucinations almost overnight. No image, no claim.
ChatGPT auth as the default zero-cost path. Anyone with a Plus subscription runs this for free. The Codex endpoint streams cleanly. The most important architectural decision in the whole submission.
Surfacing AI activity in a live ticker. The user sees the scrape chain trying og:image, then JSON-LD, then a Twitter card fallback. Transparency is what makes the system feel real instead of magical.

⚠  What did not work first try

The OAuth poll endpoint URL was wrong. First version polled /api/accounts/deviceauth/poll; the correct endpoint is /api/accounts/deviceauth/token. Symptom: user entered the code on auth.openai.com and the cockpit never noticed. Fix was one line.
The provider modal was clipped off-screen. Sticky header uses backdrop-filter: blur, which creates a new containing block, so fixed inset-0 was relative to the header, not the viewport. Fix: render the modal in a React portal to document.body.
The activity log left a stale "thinking" spinner after the run finished. The thinking event was emitted at 0s before the API call started, and stayed in the log forever. Fix: ticker hides any thinking event whose requestId has been superseded by a later event.
Auto-fill button took too long to ship. First brief form was technically complete but felt like a tax. Adding "paste a URL or type a brand name and the cockpit fills the rest" was the change that stopped first-time bouncing.

↻  What I would iterate

Vision on Steps 2 and 4. Today the style extractor reads page metadata + competitor URL strings and uses gpt-5.4's training knowledge of those brands. Adding a vision call that actually looks at the fetched competitor images would tighten palette extraction for unknown brands. Route this leg to a BYO Gemini or Anthropic key.
Per-image regen in Step 5 with explicit edit-instructions. Today Step 5 is a pass-through. Next iteration takes the Step 4 prompt + an uploaded reference image, runs Flux Kontext with a small inpaint mask, and lets the user "remove the chair" or "make the headline bigger" without rewriting the whole prompt.
A second model in parallel for cross-validation. Run Claude 4.7 in parallel with gpt-5.4 on the concept brief stage. If models agree, ship. If they disagree, surface both for human pick. Cheap insurance against single-model bias.
Approve-and-lock gate per step. Today the user can keep regenerating downstream artefacts even after accepting Step 2. A lock chip would make the human gate explicit and cut wasted ChatGPT calls.

One subscription, no designer in the loop. A brand ships an Amazon A+ panel set or a Meta dynamic-creative batch on the same Tuesday the brief lands.
